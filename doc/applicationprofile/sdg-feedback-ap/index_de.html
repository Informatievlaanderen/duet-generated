Dieses Dokument beschreibt ein Anwendungsprofil, in diesem Fall TertiaryEducationDiplomaEvidence-feedback-AP-DE. Das Anwendungsprofil spezifiziert ein semantisches Datenmodell, welches eine Reihe von Anwendungsfällen abdeckt.
