Appplicatieprofiel Generiek-basis

This document describes an application profile, in this case Adressenregister. The application profile specifies a semantic data model covering a set of usecases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

Summary

Status of this document

This application profile has the status Kandidaat Standaard publised at niet-bepaald.

Information about the process and the decisions involved in the creation of this specification are consultable at the registry of standards.

License

Conformiteit

An implementation is conformant with this application profile if it satifies these rules.

Overview

This document describes the usage of the following entities for a correct usage of the Application Profile:
| Adresseerbaar Object | Adresuitbreiding | Belgisch Adres | Gemeente | Gemeentenaam | Ligplaats | Locatie | Postinfo | Standplaats | Straatnaam |

This document describes the usage of the following datatypes for a correct usage of the Application Profile:
| Adreslocator | Adresvoorstelling | Geografische Naam | Geografische Positie | Locatieaanduiding | Locatienaam |
