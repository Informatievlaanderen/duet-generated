Dieses Dokument beschreibt ein Anwendungsprofil, in diesem Fall Core Criterion and Core Evidence Vocabulary. Das Anwendungsprofil spezifiziert ein semantisches Datenmodell, welches eine Reihe von Anwendungsfällen abdeckt. Die Spezifikation besteht aus Begriffen mit ihren Einschränkungen (Kardinalitäten, Codelisten, Nutzungsrichtlinien).
