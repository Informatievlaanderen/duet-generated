DUETLogo Vocabularies, Applicationprofiles and codelists
Semantische afstemming zorgt er voor dat informatiemodellen gebruik maken van eenduidige termen waar een bijbehorende definitie kan voor opgezocht worden. Dit komt interoperabiliteit tussen toepassingen ten goede door dubbelzinnigheid en onduidelijkheid te elimineren. Daarnaast zorgt het voor breder gedragen informatiemodellen door silodenken te vermijden. Ontdek hier de beschikbare vocabularia en applicatieprofielen, alsook codelijsten.

  • Application profiles
  • Vocabularies
  • Codelists

Vocabularia on this page are open semantic information models. They offer a terminology basis to exhange data in the context of the DUET ecosystem.

Application Profiles are specifications in the context of a defined application context. They are based on the terms defined in the DUET vocabularies and external vocabularies. Application profiles express more precise semantics for terms in the application context. They express cardinalities, usage notes, or to-be-used codelists.


The vocabularia and application profiles are setup according to the principles of the Semantic Web and expressed therefore according to the Resource Description Framework (RDF). This makes them out-of-the box applicable for Linked Data applications.
